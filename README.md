# Jobby App

E-commerce that allow users to purchase different types of furniture.

## Table of contents
* [General info](#general-info)
* [Technologies](#technologies)
* [Setup](#setup)

## General info
The app allows the users to select the items they prefer from the catalogue and add to the cart. The overall user experience has been made more pleasant thanks to the possibility to filter all products by **room**, **brand**, **name**, **color** Â and **price**. The users have full control of their carts and can increase or decrease the number of items and clear all furniture.
	
## Technologies
The project is created with:
* react: 18.2,
* react-router-dom: 6.3,
* react-icons: 4.3.1,
* axios : 0.27.2,
* styled-components: 5.3.5,
	
## Setup
To run this project, install it locally using npm:

```
$ cd ../lorem
$ npm install
$ npm start
```
